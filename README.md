# HTML

Hypertext Markup Language. Digunakan untuk menyajikan konten dan struktur dari halaman.

**Referensi:**
- www.w3.org
- www.evolutionoftheweb.com

## Tag pada HTML

Tag yang bisa disimpan di bagian `<head></head>`
- Judul Halaman
  
  `<title></title>`
- CSS

  `<style></style>`
- JavaScript

  `<script></script>`
- Metadata

  `<meta></meta>`

Tag yang bisa disimpan di body `<body></body>`
- Teks

  `<h1> <h2> <h3> <h4> <h5> <h6> <p> ...`
- Pendukung teks

  `<br> <hr> <em> <strong>`
- Gambar

  `<img>`
- Hyperlink

  `<a>`
- List (bullets & numbering)

  `<ul> <ol> <li> <dl> <dt> <dd>`
- Tabel

  `<table> <thead> <tbody>`
- Form

  `<form> <input> <select> <button>`
- Script

  `<script>`
- Object

  `<object>`
- Grouping

  `<div><span>`
- Comment

  `<!--- --->`
